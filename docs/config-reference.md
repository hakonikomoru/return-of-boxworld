# CONFIG 設定リファレンス

> **注意:** ここでいう「設定」は、OS の**環境変数**（`.env` や `export`）ではありません。  
> ゲームの挙動を変える **`behavior_packs/robw_behavior/scripts/main.js` 先頭の `CONFIG` オブジェクト** のことです。  
> 値を変えたらワールドの再読み込み（またはゲーム再起動）で反映されます。

仕様変更後は `npm run sync:project-docs` で `docs/project-sync.md` のゲーム内ルールも同期してください。

## ゲーム時間・通知

| 設定名 | 型 | 例 | 何を決めるか |
|--------|-----|-----|----------------|
| `GATE_OPEN_MINUTES` | 数値（分） | `5` | **1 ラウンドの長さ**（実時間）。`start` から何分間、捕獲・納品ができるか。ポーズ中も経過する。 |
| `TIME_NOTIFY_INTERVAL_SECONDS` | 数値（秒） | `60` | **残り時間をチャット通知する間隔**。60 なら「あと 4:00」のように 1 分ごとに表示。 |
| `UI_BROADCAST_MAX_PER_10_SECONDS` | 数値（件） | `4` | **`chat` 時**の10秒あたり全員チャット上限。開始・終了・残り警告は優先表示。 |
| `UI_BROADCAST_DISPLAY` | 文字列 | `actionbar` | 全員通知の見た目。`actionbar`=画面上部に**1行だけ**（次で上書き）。`chat`=左上に積む従来表示。 |
| `UI_PLAYER_MESSAGE_DISPLAY` | 文字列 | `actionbar` | 個人向けメッセージ（進捗・戻った場所など）。未指定時は `UI_BROADCAST_DISPLAY` と同じ。 |
| `UI_BROADCAST_DEDUPE_SECONDS` | 数値（秒） | `30` | **`chat` 時**、同じ文言の再送を抑える秒数（`0` で無効）。 |
| `UI_BROADCAST_SEQUENCE_STEP_TICKS` | 数値（tick） | `60` | **`actionbar` 時**、ランキングなど複数行を順に出す間隔（60≒3秒）。 |

## 捕獲（骨を振る）

| 設定名              | 型               | 例               | 何を決めるか                                                                                                 |
| ------------------- | ---------------- | ---------------- | ------------------------------------------------------------------------------------------------------------ |
| `PROTECTION_RADIUS` | 数値（ブロック） | `4`              | **骨で右クリックしたとき、どれだけ近くの動物を対象にするか**。大きいほど遠くのオオカミ／牛なども捕獲できる。 |
| `PROTECT_ITEM`      | アイテム ID      | `minecraft:bone` | **捕獲に使うアイテム**。通常は骨のまま変更不要。                                                             |
| `BONES_PER_CAPTURE` | 数値（個）       | `1`              | **捕獲 1 回で消費する骨**。`0` で消費なし。                                                                  |

## マップ・座標（フォールバック）

通常は **start 時に選んだステージ**（`/locate` で探した構造物付近、またはホスト足元）がラウンドの中心になります。  
ゲーム未起動時や座標が取れないときだけ、下の値が使われます。

| 設定名 | 型 | 例 | 何を決めるか |
|--------|-----|-----|----------------|
| `ROUND_STAGES` | オブジェクト配列 | `{ id, label, locate, dimension }` | **構造物へ移動**の一覧。`locate` は `/locate structure` の ID。**登録済み座標**があれば即 TP。未登録時は「構造物を探す」→移動→「現在地をステージ登録」。**チート ON 推奨**。 |
| `STRUCTURE_LOCATE_SHORTCUT_STAGE_IDS` | 文字列配列 | `village` 等 | **構造物を探す**（手動用 `/locate` 案内）／**現在地をステージ登録**に出す `ROUND_STAGES` の `id` 一覧。 |
| `STAGE_LOCATE_PREFETCH_ON_JOIN` | 真偽値 | `false` | ホスト入場後、構造物座標を裏でプリフェッチする（API 未対応環境では効果が薄い）。 |
| `STAGE_LOCATE_PREFETCH_SCAN_BLOCKS` | 数値（ブロック） | `8192` | プリフェッチ時の `getGeneratedStructures` スキャン半径。 |
| `STAGE_LOCATE_MAX_SCAN_BLOCKS` | 数値（ブロック） | `8192` | （レガシー probe 用）構造物スキャン最大半径。 |
| `STAGE_LOCATE_SCAN_STEP` | 数値（ブロック） | `128` | （レガシー probe 用）スキャン間隔。 |
| `STAGE_LOCATE_MAX_PROBES` | 数値 | `48` | （レガシー probe 用）最大試行数。 |
| `STAGE_LOCATE_PROBE_WAIT_TICKS` | 数値 | `10` | （レガシー probe 用）chunk 待ち tick。 |
| `BOX_GATE` | オブジェクト | `{ x, y, z, radius }` | **ラウンド中心のフォールバック座標**。`x,z` はブロック座標、`y` は足元の高さの目安、`radius` はスポーン範囲の参考値。 |
| `SUBMISSION_CHEST` | オブジェクト | `{ x, y, z }` | **納品チェストのフォールバック座標**。通常は起動者の足元に自動設置されるため、テスト用以外はあまり触らない。 |
| `CHEST_CLEANUP_RADIUS` | 数値（ブロック） | `12` | **start 時に、足元付近の既存チェスト・樽などを消す水平半径**。 |
| `CHEST_CLEANUP_VERTICAL_RANGE` | 数値（ブロック） | `10` | **地面から上下に探して撤去する高さ**。 |
| `ROUND_ITEM_CLEANUP_RADIUS` | 数値（ブロック） | `0` | **落とし物を消す半径**。`0` でスポーン範囲に合わせる（おおよそ 70）。 |
| `DROPPED_ITEM_ENTITY_TYPES` | ID の配列 | `minecraft:item` | **消す落とし物エンティティ**。 |
| `SUBMISSION_CHEST_BLOCK_TYPES` | ID の配列 | チェスト等 | **「既存の収納ブロック」として撤去するブロック種類**。通常は変更不要。 |
| `PROTECT_SUBMISSION_CHEST` | 真偽 | `true` | **ラウンド中**、自動設置した納品チェストの破壊を防ぐ（通常モード）。 |
| `GATE_SUMMON_OFFSET_Y` | 数値 | `0` | **互換用（現在ほぼ未使用）**。テレポート廃止に伴い予約。 |

## ハコイヌ100匹チャレンジ (`CONFIG.BOX100`)

通常モードとは別のタイムアタック。`!robw start box100` またはメニュー「ハコイヌ100匹チャレンジ」で開始。

| 設定名 | 型 | 例 | 何を決めるか |
|--------|-----|-----|----------------|
| `WOLF_COUNT` | 数値 | `100` | 各部屋のオオカミ数。負荷テスト時は `10` などに下げる。 |
| `BONE_COUNT` | 数値 | `120` | 開始時に全員へ配る骨の数。 |
| `TIME_LIMIT_SECONDS` | 数値 | `180`（3分） | 制限時間（秒）。 |
| `ROOM_SIZE` / `ROOM_HEIGHT` | 数値 | `30` / `10` | ガラス箱の幅・高さ。 |
| `ROOM_GAP` | 数値 | `10` | 部屋と部屋のあいだの空き（ブロック）。 |
| `ROOM_GRID_SPACING` | 数値 | `40` | グリッド配置の間隔（通常 `ROOM_SIZE + ROOM_GAP`）。 |
| `ROOM_GRID_COLUMNS` | 数値 | `2` | 横方向の部屋数。**4人まで 2×2**（■■／■■）。5〜6人は列数を自動拡張。 |
| `PROTECT_ROOM_STRUCTURE` | 真偽 | `true` | **ラウンド中**、外殻ガラスとシュルカーの破壊を防ぐ。 |
| `SKY_ARENA_ENABLED` | 真偽 | `true` | **天空に部屋を出す**（地上の建造物を壊さない）。`false` でホスト足元の高さ。 |
| `SKY_ARENA_BASE_Y` | 数値 | `200` | 天空アリーナの床 Y。水平位置はホストの X/Z。 |
| `MAX_PLAYERS` | 数値 | `6` | 同時参加の上限。 |
| `NIGHT_VISION_ENABLED` | 真偽 | `true` | **ラウンド中だけ暗視**を付与（箱内が暗いため）。終了・停止で解除。 |
| `RANDOMIZE_ROOM_COLORS` | 真偽 | `true` | **毎回**箱の色をランダム割り当て。`false` で参加順に固定。 |
| `COLORS` | 配列 | `glass`・`shulker` ID | 抽選候補の色。外殻はシュルカーと同系の**色付きガラス**。 |

終了時はガラス箱を削除し、オオカミ・シュルカー・一時アイテムを片付けます（`/fill` にはチートONが必要）。

## 開始時の配布・スポーン

| 設定名                             | 型               | 例     | 何を決めるか                                                                                                                 |
| ---------------------------------- | ---------------- | ------ | ---------------------------------------------------------------------------------------------------------------------------- |
| `START_GIVE_BONES`                 | 数値（個）       | `12`   | **ラウンド開始時に全員へ渡す骨の数**。いったん所持分を消してからこの数で配る。                                               |
| `BONES_PER_HAKOINU_DELIVERY`       | 数値（個/枚）    | `2`    | **ハコイヌの毛皮を納品したとき、1 枚あたりもらえる骨**。                                                                     |
| `BONES_PER_WRONG_ANIMAL_DELIVERY`  | 数値（個/枚）    | `4`    | **別種の毛皮を納品したとき、1 枚あたりもらえる骨**。                                                                         |
| `START_SPAWN_HAKOINU`              | 数値（匹）       | `90`   | **出現させるオオカミの目標数**（地形で失敗する分は `SPAWN_ATTEMPTS_PER_ENTITY` で再試行）。Switch 等で重いときは `70` 前後。 |
| `START_SPAWN_PENALTY_ANIMALS`      | 数値（匹）       | `20`   | **出現する別種の数**。                                                                                                       |
| `SPAWN_ATTEMPTS_PER_ENTITY`        | 数値（回）       | `8`    | **1 匹あたりのスポーン試行回数**（高さを変えて複数座標を試す）。                                                             |
| `SPAWN_MIN_DISTANCE`               | 数値（ブロック） | `6`    | **チェスト付近を空ける**最短距離。                                                                                           |
| `SPAWN_MAX_DISTANCE`               | 数値（ブロック） | `40`   | **探索の最遠距離**の目安。狭めにすると遭遇しやすい。                                                                         |
| `SPAWN_REPLENISH_ENABLED`          | 真偽             | `true` | **ラウンド中に減った分を目標数まで補充**するか。                                                                             |
| `SPAWN_REPLENISH_INTERVAL_SECONDS` | 数値（秒）       | `4`    | **補充チェックの間隔**。短いほどすぐ戻るが負荷増。                                                                           |
| `SPAWN_REPLENISH_MAX_PER_CHECK`    | 数値（匹）       | `6`    | **1 回のチェックで増やす上限**（ハコイヌ＋別種の合計）。                                                                     |
| `SPAWN_NEAR_PLAYERS`               | 真偽             | `true` | **プレイヤー付近に湧かせる**（false ならゲート中心のみ）。                                                                   |
| `SPAWN_NEAR_PLAYER_MIN_DISTANCE`   | 数値（ブロック） | `8`    | プレイヤー足元を空ける最短距離。                                                                                             |
| `SPAWN_NEAR_PLAYER_MAX_DISTANCE`   | 数値（ブロック） | `18`   | プレイヤーから湧く最遠距離（歩いてすぐ会える範囲）。                                                                         |
| `SPAWN_MIN_NEAR_PLAYER`            | 数値（匹）       | `4`    | **各プレイヤー周辺に最低キープする動物数**（ハコイヌ＋別種）。                                                               |
| `SPAWN_ON_CAPTURE_ENABLED`         | 真偽             | `true` | **捕獲直後**にそのプレイヤー付近へ即スポーン。                                                                               |
| `SPAWN_ON_CAPTURE_COUNT`           | 数値（匹）       | `1`    | 捕獲 1 回あたりに近くへ出すハコイヌ数。                                                                                      |

## 捕獲アイテム（毛皮）・動物の種類

| 設定名                                      | 型          | 例                      | 何を決めるか                                                  |
| ------------------------------------------- | ----------- | ----------------------- | ------------------------------------------------------------- |
| `RETURN_BOX_ITEM`                           | アイテム ID | `minecraft:rabbit_hide` | **捕獲すると手に入るアイテムの見た目**（茶色の毛皮）。        |
| `RETURN_BOX_DISPLAY_NAME`                   | 文字列      | `捕獲した毛皮`          | **プレイヤーに見える名前**。ハコイヌも別種も同じ。            |
| `RETURN_BOX_KIND_PROPERTY`                  | 文字列      | `robw:return_kind`      | **正誤を記録する内部 ID**（画面には出ない）。通常は変更不要。 |

| `HAKOINU_ENTITY_TYPES`                      | ID の配列   | `minecraft:wolf`        | **ハコイヌとして扱う Mob**。MVP はオオカミ 1 種。             |
| `PENALTY_ANIMAL_TYPES`                      | ID の配列   | 牛・豚…                 | **別種としてスポーン／誤捕獲ペナルティの対象になる Mob**。    |

## スコア（帰還ポイント）

| 設定名                   | 型         | 例             | 何を決めるか                                                                                                                                                                             |
| ------------------------ | ---------- | -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `POINTS_PER_BOX`         | 数値（pt） | `1`            | **ハコイヌの毛皮を 1 枚納品したときの得点**。                                                                                                                                            |
| `POINTS_WRONG_ANIMAL`    | 数値（pt） | `-2`           | **別種の毛皮を 1 枚納品したときの減点**。マイナスなのでそのまま負の数で書く。                                                                                                            |
| `POINTS_CHEST_JUNK_ITEM` | 数値（pt） | `-1`           | **毛皮以外を納品チェストに入れたときの減点（1 個あたり）**。入れたものは吸収（消去）される。**操作時計・骨（`PROTECT_ITEM`）は除く**。                                                   |
| `POINTS_HAKOINU_KILL`    | 数値（pt） | `-10`          | **ラウンド中にハコイヌ（オオカミ）を倒したときの減点**（加害プレイヤーへ）。                                                                                                             |
| `POINTS_HAKOINU_HIT`     | 数値（pt） | `-1`           | **ハコイヌへ与えたダメージ 1 回あたりの減点**（加害プレイヤーへ）。                                                                                                                      |
| `POINTS_PLAYER_DEATH`    | 数値（pt） | `-10`          | **ラウンド中にプレイヤーが死亡したときの減点**。致死直前に開始地点へ戻し、個人スポーンも開始地点に設定（リスポーン画面を出さない／押しても開始地点）。骨は `START_GIVE_BONES` 本に補充。 |
| `SCORE_OBJECTIVE`        | 文字列     | `return_point` | **スコアボードの ID**。他アドオンと被る場合だけ変更。                                                                                                                                    |

| 設定名 | 型 | 例 | 何を決めるか |
|--------|-----|-----|----------------|

## 天候・時刻（昼固定）

| 設定名                         | 型           | 例       | 何を決めるか                                                               |
| ------------------------------ | ------------ | -------- | -------------------------------------------------------------------------- |
| `LOCK_DAYTIME`                 | true/false   | `true`   | **昼・晴天を固定するか**。`false` にするとマイクラ本来の昼夜・天候に戻る。 |
| `DAY_TIME_OF_DAY`              | 数値（tick） | `6000`   | **固定する時刻**。`0` 日の出、`6000` 真昼、`12000` 日暮れ、`18000` 夜。    |
| `WEATHER_CLEAR_DURATION_TICKS` | 数値（tick） | `100000` | **晴天を維持する長さ**。20 tick ≒ 1 秒。                                   |

## 開始演出（3・2・1）

| 設定名                       | 型           | 例     | 何を決めるか                                                                                                                              |
| ---------------------------- | ------------ | ------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
| `START_COUNTDOWN_ENABLED`    | true/false   | `true` | **start 時に 3・2・1 カウントダウンを出すか**。                                                                                           |
| `START_COUNTDOWN_STEP_TICKS` | 数値（tick） | `20`   | **カウント 1 歩の長さ**。開始・終了演出で共通。                                                                                           |
| `END_COUNTDOWN_ENABLED`      | true/false   | `true` | **終了時に 3・2・1 → ゲート閉鎖の演出**。時間切れ時は**残り 3・2・1 秒と同期**（その後すぐ閉鎖）。手動 stop 時は従来どおり約 4 秒の演出。 |

## 残り時間表示（HUD）

| 設定名                    | 型         | 例           | 何を決めるか                                                                        |
| ------------------------- | ---------- | ------------ | ----------------------------------------------------------------------------------- |
| `SHOW_REMAINING_TIME_HUD` | true/false | `true`       | **ゲート開放中に残り時間などを常時表示**するか。オフで非表示。                      |
| `TIMER_HUD_USE_SIDEBAR`   | true/false | `true`       | **true** = 画面右サイドバー（既定）。**false** = アクションバー中央下（数字なし）。 |

## よくある調整例

| やりたいこと                 | 触る設定                                                                                                       |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------- |
| もっと短い 1 試合            | `GATE_OPEN_MINUTES` を `3` など                                                                                |
| もっと広いマップにしたい     | `SPAWN_MAX_DISTANCE` を `64`〜`80`（5 分なら 72 程度まで）                                                     |
| もっと狭く密集させたい       | `SPAWN_MAX_DISTANCE` を `32`、匹数を増やす                                                                     |
| 捕獲後も常にウルフがいる感じ | `SPAWN_MIN_NEAR_PLAYER` を `6`、`SPAWN_ON_CAPTURE_COUNT` を `2`                                                |
| 補充で重い                   | `START_SPAWN_HAKOINU` を `70`、`SPAWN_REPLENISH_MAX_PER_CHECK` を `4`、または `SPAWN_REPLENISH_ENABLED: false` |
| 納品で骨がたくさんもらえる   | `BONES_PER_HAKOINU_DELIVERY` を `3` など                                                                       |
| 誤捕獲を厳しく               | `POINTS_WRONG_ANIMAL` を `-3` など                                                                             |
| 倒すと厳しく                 | `POINTS_HAKOINU_KILL` を `-15` など                                                                            |
| カウントダウンを速く         | `START_COUNTDOWN_STEP_TICKS` を `10` など                                                                      |
| 夜のままにしたい             | `LOCK_DAYTIME: false`                                                                                          |
